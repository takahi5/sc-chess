# 運用コマンドまとめ

開発・運用時によく使うコマンドを以下に整理しています。すべてプロジェクトルートで実行してください。

## セットアップ
- `pnpm install`: 依存関係をインストールします。

## 開発サーバーとプラットフォーム起動
- `pnpm start`: Expo開発サーバーを起動します。
- `pnpm start -- --clear`: バンドラのキャッシュをクリアして開発サーバーを再起動します。
- `pnpm android`: Androidエミュレーターまたは実機でExpoアプリを立ち上げます。
- `pnpm ios`: iOSシミュレーターまたは実機でExpoアプリを立ち上げます。
- `pnpm web`: Webターゲットでアプリを起動します。

## 品質チェック
- `pnpm lint`: ESLintを実行し、静的解析によるコードチェックを行います。

## プロジェクトメンテナンス
- `pnpm reset-project`: 初期状態の`app/`ディレクトリを再生成します。実行前に作業内容のバックアップを推奨します。
